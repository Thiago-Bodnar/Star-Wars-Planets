
# Star Wars Planets

Uma aplicação que lista todos os planetas conhecidos do universo de Star Wars, sendo possível aplicar diversos filtros de ordem e de busca com critérios diferentes.

## Demonstração

![starwars](https://user-images.githubusercontent.com/94481341/165177490-7913bf94-509a-49af-be6c-6ea21f727e5d.gif)



## Aprendizados

Aprendi bastante sobre o método filter e também sobre componentes funcionais do React, sendo o maior desafio implementar filtros em cima de outros filtros, para que funcionem em conjunto.


## Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:Thiago-Bodnar/Star-Wars-Planets.git
```

Entre no diretório do projeto

```bash
  cd Star-Wars-Planets
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```


## Stack utilizada

**Front-end:** React e Bootstrap

**Back-end:** Api dos planetas de Star Wars
